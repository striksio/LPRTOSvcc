# Laser-Prompted Real-Time Object Segmentation on Smartphones

Android client for the LPRTOS pipeline.

## Requirements
- JDK 11
- Smartphone with a back camera (Xiaomi Mi 9 used originally)

## Build and run
1. Open this folder in Android Studio
2. Connect a device with USB debugging enabled
3. Run the app
4. Enter the server IP in the app, tap Start Streaming

Default streaming port: 8765
